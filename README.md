# Portfolio

Static portfolio scaffold with a selector landing page plus a base portfolio layout.

## Files

- `index.html`: landing page for choosing site mode (standard or themed placeholders).
- `standard.html`: base portfolio layout with placeholder content.
- `styles.css`: shared styling.
- `main.js`: lightweight interactions.

## Run locally

Open `index.html` in a browser, or use a simple static server:

```powershell
npx serve .
```
